 Название Telegram-бота: Music_bot

Краткое описание:  
Это Telegram-бот, который позволяет найти музыку которую ищет. 

# Основные возможности
- Поиск треков через API Last.fm  
- Получение информации об исполнителях  
- Добавить в избронную 

## Технологии
- Python 3.12  
- [aiogram](https://docs.aiogram.dev/) (асинхронный фреймворк для Telegram)  
- aiohttp — для HTTP-запросов  
- dotenv — для загрузки переменных окружения
